# GestãoPro

## Sobre o projeto

GestãoPro é a aplicação de gestão da Pronutrir.

## Como rodar localmente

O único requisito é ter Node.js & npm instalados - [instale com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Passo 1: Clone o repositório.
git clone https://github.com/Pronutrir/GestaoPro.git

# Passo 2: Entre no diretório do projeto.
cd GestaoPro

# Passo 3: Instale as dependências.
npm i

# Passo 4: Inicie o servidor de desenvolvimento.
npm run dev
```

## Tecnologias utilizadas

Este projeto é construído com:

- Next.js
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
